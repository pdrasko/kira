MIDI Virtual Keyboard

Proof of concept — connects a MIDI-to-USB synthesizer to Chrome's Web MIDI API and mirrors what you play on an on-screen keyboard.

Web MIDI isn't available in this browser.
Web MIDI (navigator.requestMIDIAccess) only works in Chrome/Edge on desktop and Android. It is not supported by any browser on iOS/iPadOS (including Chrome for iOS, which uses Apple's WebKit engine under the hood). If you're testing from a phone, use an Android device.
